1
INSERT INTO "oeuvres" VALUES (NULL,

3
"Essai philosophique", 2018, "L'aventure") ;

Question

Solution
1
SELECT Nom_Auteur, titre, annee_publication FROM Oeuvres
2
WHERE annee_publication >= (SELECT annee_publication FROM Oeuvres WHERE titre = 'Lettres persanes')
3
ORDER BY annee_publication ;

Question

Donner le nombre d’œuvres de Molière.

Solution
1
SELECT COUNT(DISTINCT Titre) FROM Oeuvres WHERE Nom_auteur = 'Molière' ;
SELECT COUNT(DISTINCT Titre) FROM Oeuvres WHERE Nom_auteur = 'Molière' ;

Le mot clé DISTINCT peut servir à différencier des ouvrages référencés plusieurs fois.

Question

Afficher les nationalités des auteurs et le nombre d'ouvrages d'auteurs de ces nationalités. L'affichage se fera dans l'ordre décroissant.

Solution
1
SELECT Nationalite_Auteur, COUNT(*) AS c FROM oeuvres GROUP BY Nationalite_Auteur ORDER BY c DESC;
