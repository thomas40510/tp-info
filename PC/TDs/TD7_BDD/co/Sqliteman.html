Logiciel simple de gestion de bases de données

Logiciel

Sqliteman est un logiciel open source et gratuit qui permet de gérer des bases de données et de tester des requêtes SQL.

Le fichier ci-dessous permet d'installer ce logiciel.

Fichier à décompresser, puis lancer l'application sqliteman

Prise en main

Ouvrir le répertoire Ressources Pedagogiques\Informatique pour tous\Eleves, et copier le répertoire sqliteman dans son répertoire Mes documents. Ouvrir ensuite la copie du répertoire et exécuter le fichier sqliteman.exe pour démarrer le logiciel.

Créer une nouvelle base de données (menu Fichier), en la sauvegardant dans un sous-répertoire de son répertoire Mes documents.

Créer ses tables. Il est possible d'utiliser l'interface graphique avec les menus déroulants (clic droit) sur l'arborescence de gauche, ou d'écrire directement les requêtes SQL dans la fenêtre de droite. Un clic sur la flèche verte dans la barre de menu exécutera les requêtes.

On peut faire un clic droit sur Tables dans l'arborescence de la fenêtre de droite et choisir Créer une Table, ou taper une requête sur le modèle ci-dessous dans la fenêtre en haut à droite.

CREATE TABLE "Livres" (

    "titre" TEXT,

Ensuite, il faut créer les différents enregistrements. Pour cela, il faut double-cliquer sur la table créée. La fenêtre centrale de droite devient active, et il est possible de créer son premier enregistrement en cliquant sur le premier icône de la barre de cette fenêtre.

Ne pas renseigner la première colonne qui doit s'auto incrémenter.

Attention, tant que vous n'avez pas validé votre travail avec le quatrième icône de la barre de menu précédente, vos enregistrements ne sont pas insérés dans votre base de données. C'est le principe d'une architecture client - serveur.
