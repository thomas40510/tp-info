Gestion pédagogique des classes préparatoires

Voici la base de données qui va servir de support à cette partie.

Jointure à deux tables.

Question

Afficher les noms et villes des lycées de Loire-Atlantique (département 44).

Solution
1
SELECT Lycees.nom, Villes.nom FROM Lycees JOIN Villes ON Lycees.id_ville = Villes.id WHERE Villes.departement = '44' ;
SELECT Lycees.nom, Villes.nom FROM Lycees JOIN Villes ON Lycees.id_ville = Villes.id WHERE Villes.departement = '44' ;

Question

Afficher les noms des lycées situés dans une ville de moins de 250 000 habitants.

Solution
1
SELECT L.nom FROM Lycees AS L JOIN Villes AS V ON L.id_ville = V.id WHERE population <= 250000 ;
SELECT L.nom FROM Lycees AS L JOIN Villes AS V ON L.id_ville = V.id WHERE population <= 250000 ;

Question

Afficher le nombre de lycées situés à Nantes.

Solution
1
SELECT COUNT(*) FROM Lycees, Villes WHERE Lycees.id_ville = Villes.id AND Villes.nom = 'Nantes' ;
SELECT COUNT(*) FROM Lycees, Villes WHERE Lycees.id_ville = Villes.id AND Villes.nom = 'Nantes' ;

Question

Afficher le département de la ville du lycée Bergson.

Solution
1
SELECT V.departement FROM Villes AS V, Lycees AS L WHERE V.id = L.id_ville AND L.nom = 'Bergson' ;
SELECT V.departement FROM Villes AS V, Lycees AS L WHERE V.id = L.id_ville AND L.nom = 'Bergson' ;

Jointure à trois tables et plus.

Question

Afficher les noms des lycées et filières proposées dans chacun d'entre eux.

Solution
1
SELECT L.nom, F.nom FROM Lycees AS L, Filieres AS F, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil GROUP BY L.nom ;
SELECT L.nom, F.nom FROM Lycees AS L, Filieres AS F, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil GROUP BY L.nom ;

Question

Afficher les noms des lycées proposant une PCSI.

Solution
1
SELECT L.nom FROM Lycees AS L, Filieres AS F, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil AND F.nom = 'PCSI' ;
SELECT L.nom FROM Lycees AS L, Filieres AS F, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil AND F.nom = 'PCSI' ;

Question

Afficher les filières (avec lycées et villes correspondantes) proposées dans la Sarthe (département 72).

Solution
1
SELECT F.nom, L.nom, V.nom FROM Lycees AS L, Filieres AS F, Villes AS V, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil AND L.id_ville = V.id AND V.departement = 72 GROUP BY F.nom;
SELECT F.nom, L.nom, V.nom FROM Lycees AS L, Filieres AS F, Villes AS V, Rel_Lyc_Fil AS RLF WHERE L.id = RLF.id_lyc AND F.id = RLF.id_fil AND L.id_ville = V.id AND V.departement = 72 GROUP BY F.nom;

Question

Afficher les noms des villes où l'on peut trouver une filière PSI.

Solution
1
SELECT V.nom FROM Villes AS V, Filieres AS F, Lycees AS L, Rel_Lyc_Fil AS RLF WHERE F.nom = 'PSI' AND F.id = RLF.id_fil AND RLF.id_lyc = L.id AND L.id_ville = V.id ;
SELECT V.nom FROM Villes AS V, Filieres AS F, Lycees AS L, Rel_Lyc_Fil AS RLF WHERE F.nom = 'PSI' AND F.id = RLF.id_fil AND RLF.id_lyc = L.id AND L.id_ville = V.id ;

Question

Afficher les noms des filières proposées dans les villes de moins de 250 000 habitants.

Solution
1
SELECT F.nom FROM Filieres AS F, Rel_Lyc_Fil AS RLF, Lycees AS L, Villes AS V WHERE V.population <= 250000 AND V.id = L.id_ville AND L.id = RLF.id_lyc AND RLF.id_fil = F.id ;
SELECT F.nom FROM Filieres AS F, Rel_Lyc_Fil AS RLF, Lycees AS L, Villes AS V WHERE V.population <= 250000 AND V.id = L.id_ville AND L.id = RLF.id_lyc AND RLF.id_fil = F.id ;
