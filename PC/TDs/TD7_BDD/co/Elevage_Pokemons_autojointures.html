Élevage de Pokemons

Pour illustrer les auto-jointures, voici une table utilisée pour gérer un petit élevage de Pokemons.

La table Pokemons possède les attributs suivants : id (la clé primaire), nom, sexe, niveau, pv, taille, poids, valeur, anneenaissance, idmere, idpere, anneevente et anneemort.

Vous pouvez tester vos requêtes avec Sqliteman et ce fichier.

Gros élevage de Pokemons

Question

Afficher les noms des enfants du pokemon femelle "Ludicolo", ainsi que les noms de leurs pères respectifs.

Solution
1
SELECT E.nom, P.nom FROM Pokemons AS E JOIN Pokemons AS P ON P.id = E.idpere WHERE E.idmere = (SELECT id FROM Pokemons WHERE nom = 'Ludicolo') ;
SELECT E.nom, P.nom FROM Pokemons AS E JOIN Pokemons AS P ON P.id = E.idpere WHERE E.idmere = (SELECT id FROM Pokemons WHERE nom = 'Ludicolo') ;

Question

Afficher les noms des enfants et petits-enfants de "Ludicolo".

Solution
1
SELECT E.nom, PE.nom FROM Pokemons AS E JOIN Pokemons AS PE ON (PE.idmere = E.id OR PE.idpere = E.id) WHERE E.idmere = (SELECT id FROM Pokemons WHERE nom = 'Ludicolo') ;
SELECT E.nom, PE.nom FROM Pokemons AS E JOIN Pokemons AS PE ON (PE.idmere = E.id OR PE.idpere = E.id) WHERE E.idmere = (SELECT id FROM Pokemons WHERE nom = 'Ludicolo') ;

Question

Afficher les noms des pokemons frères entiers.

Solution
1
SELECT F1.nom FROM Pokemons AS F1 JOIN Pokemons AS F2 ON (F1.idpere = F2.idpere AND F1.idmere = F2.idmere) WHERE (F1.sexe = 'M' AND F2.sexe = 'M') ;
SELECT F1.nom FROM Pokemons AS F1 JOIN Pokemons AS F2 ON (F1.idpere = F2.idpere AND F1.idmere = F2.idmere) WHERE (F1.sexe = 'M' AND F2.sexe = 'M') ;

Question

Afficher le nombre de pokemons morts avant leurs deux parents (ou en même temps).

Solution
1
SELECT COUNT(*) FROM Pokemons AS E JOIN Pokemons AS P ON E.idpere = P.id JOIN Pokemons AS M ON E.idmere = M.id WHERE E.anneemort <= P.anneemort AND E.anneemort <= M.anneemort ;
SELECT COUNT(*) FROM Pokemons AS E JOIN Pokemons AS P ON E.idpere = P.id JOIN Pokemons AS M ON E.idmere = M.id WHERE E.anneemort <= P.anneemort AND E.anneemort <= M.anneemort ;
