Le chiffre Vigenère

Le chiffre de Vigenère est un chiffrement basé sur une substitution dont la clé est constituée par un texte : une lettre de l'alphabet dans le texte en clair peut être chiffrée de plusieurs manières.

Au XVIème siècle, Blaise de Vigenère fut l'un des premiers à présenter ce type de chiffrement sous la forme d'une table avec la présence d'une clé secrète.

Le chiffre de Vigenère est resté inviolé pendant plusieurs siècles.

Si l'on prend par exemple la clé « concours ». Pour crypter un texte, on code de la même manière qu'avec le code César la première lettre en utilisant le décalage qui remplace la lettre a par la lettre c, première lettre de la clé. Pour la deuxième lettre, on utilise le code qui remplace la lettre a par la lettre o. Et ainsi de suite. Pour la neuvième lettre, on reprend la clé correspondant à la première lettre.

Question

Écrire une fonction qui permette d'encoder un message passé en argument en fonction d'une clé donnée sous la forme d'un texte.

Indice

Voici ce que doit renvoyer le codage de la phrase : 'QUE J AIME A FAIRE CONNAITRE UN NOMBRE UTILE AUX SAGES'.

1
>>> codage_vigenere('QUE J AIME A FAIRE CONNAITRE UN NOMBRE UTILE AUX SAGES', 'ARCHIMEDE')
2
'DYT E RYDR P AZZHV GDHIZZJIR JH MFCSEI OOHCU NYM NZXUJ'
>>> codage_vigenere('QUE J AIME A FAIRE CONNAITRE UN NOMBRE UTILE AUX SAGES', 'ARCHIMEDE')
'DYT E RYDR P AZZHV GDHIZZJIR JH MFCSEI OOHCU NYM NZXUJ'
Solution

Voici une solution directement inspirée de la fonction codage_cesar().

def codage_vigenere(texte, cle):
2
    message = ''
3
    long = len(cle)
4
    for rang in range(len(texte)):
5
        if texte[rang] == ' ':
6
            message += ' '
7
        else:
8
            message += chr(( ord(texte[rang]) - 65 + ord(cle[rang % long])) % 26 + 65)

    return(message)
def codage_vigenere(texte, cle):

Question

Écrire la fonction qui permet le décodage du message à partir de la clé.

Solution

C'est un simple copié collé de la fonction précédente avec un facteur (-1) devant le terme dû à la clé.

def decodage_vigenere(texte, cle):
    message = ''
    long = len(cle)
    for rang in range(len(texte)):
        if texte[rang] == ' ':
            message += ' '
        else:
            message += chr(( ord(texte[rang]) - 65 - ord(cle[rang % long])) % 26 + 65)
    return(message)
