La scytale

Chez les Spartiates, la scytale, également connue sous le nom de bâton de Plutarque, était un bâton de bois utilisé pour lire ou écrire une dépêche chiffrée. Considérée comme le plus ancien dispositif de cryptographie militaire connue, elle permettait l'inscription d'un message chiffré sur une fine lanière de cuir ou de parchemin. Après avoir enroulé la ceinture sur la scytale, le message était écrit en plaçant une lettre sur chaque circonvolution. Pour le déchiffrer, le destinataire devait posséder un bâton d'un diamètre identique à celui utilisé pour l'encodage. Il lui suffit d'enrouler la scytale autour de ce bâton pour obtenir le message en clair.

def codage_scytale(texte, cle):
    '''texte est la chaîne de caractères à coder, et cle le nombre de faces du baton de Plutarque.'''
    # Division euclidienne de la longueur du texte par la périodicité du bâton
    taille = len(texte)
    quotient = taille // cle
    reste = taille % cle
    nlettres = quotient + (reste != 0)
    
    if reste != 0:
        # le texte est complété par des caractères spéciaux
        texte += ' ' * (cle - reste)
    
    # On doit écrire sur la scytale enrubannée "cle" lignes de "nlettres"
    # On peut aussi voir les choses comme une matrice contenant les lettres
    # constituées de clé lignes, et qui est lue en colonnes pour être codée.
    Lignes = []
    for l in range(cle):
        Lignes.append(texte[l * nlettres: (l + 1) * nlettres])

    tc = ''     # Texte crypté sur le ruban
    
    for icolonne in range(nlettres):
        for iligne in range(cle):
            tc += Lignes[iligne][icolonne]
    
    return(tc.strip())

def decodage_scytale_var(texte, cle):
    taille = len(texte)
    quotient = taille // cle
    reste = taille % cle
    nlettres = quotient + (reste != 0)
    
    if reste != 0:
        # le texte est complété par des caractères spéciaux
        texte += ' ' * (cle - reste)
    
    Lignes = []
    for i in range(cle): Lignes.append([])
    
    for il in range(taille):
        Lignes[il % cle].append(texte[il])
    
    tdc = ''    # Texte décrypté
    
    for lignes in Lignes:
        for lettre in lignes:
            tdc += lettre
    
    return(tdc)
