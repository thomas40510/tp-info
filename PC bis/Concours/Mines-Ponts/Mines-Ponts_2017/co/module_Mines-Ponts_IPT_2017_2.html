Préliminaires

Définir rapidement une file de voitures

Contrairement au sujet, on propose de définir l'état d'une voiture directement en binaire.

Un nombre binaire peut s'écrire par une chaîne de caractères.

Par exemple, 23 s'écrit '10111' ou '0b10111'.

Python dispose de deux fonctions intégrées pour passer du binaire au décimal et réciproquement :

1
In [1]: bin(23)
2
Out[1]: '0b10111'
In [1]: bin(23)
Out[1]: '0b10111'
1
In [2]: int('10111', 2)
2
Out[2]: 23
In [2]: int('10111', 2)
Out[2]: 23

La fonction int(chaîne de caractère, base) permet donc de renvoyer l'entier correspondant dans la base décimale.

Enfin, Python permet de convertir un entier en booléen avec la fonction intégrée bool().

1
In [3]: bool(0), bool(1), bool(5.3)
2
Out[3]: (False, True, True)
In [3]: bool(0), bool(1), bool(5.3)
Out[3]: (False, True, True)

Question

Définir une fonction convert() qui prend en argument une chaîne de caractères qui est la description en binaire de la file de voiture et qui renvoie une liste de booléens.

def egal(L1, L2):
    n1, n2 = len(L1), len(L2)
    if n1 != n2:
        return(False)
    else:
        i = 0
