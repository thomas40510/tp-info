Liste en intention

Voici à présent deux exercices pour manipuler des listes en intention.

Une liste en intention est une liste construite à partir d'une autre séquence.

Question

Q8 : (2 x 0.5 = 1 point)

Expliciter les listes L1 et L2.

1
L1 = list(range(0, 20, 3))
2
L2 = [elt for elt in L1 if elt % 2 == 0]
L1 = list(range(0, 20, 3))
L2 = [elt for elt in L1 if elt % 2 == 0]

Question

Q9 : (2 x 0.5 = 1 point)

Expliciter les listes L1, L3 et L4.

1

2
L3 = list(range(0, 20, 4))
3
L4 = [elt for elt in L1 if elt in L3]
L1 = list(range(0, 20, 3))
L3 = list(range(0, 20, 4))
L4 = [elt for elt in L1 if elt in L3]
